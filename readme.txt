Emails author when pending post is published.

== Description ==

Emails author when pending post is published. This can be useful for sites with multiple authors where editors publish content written by authors.

== Installation ==

1. Upload the `email-on-pending-post` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
